# Portrait photos

Drop the two portraits here. Until they exist, the Hero and About sections fall
back automatically to the striped placeholder (see `src/components/ui/Photo.tsx`).

| File                 | Used in            | Ratio          | Direction |
| -------------------- | ------------------ | -------------- | --------- |
| `portrait-hero.jpg`  | Hero               | 4/5 (5/4 wide) | Bright, confident upper-body portrait, warm natural light, clean warm-neutral background. Leave headroom so the "Verfügbar" badge doesn't crowd the face. |
| `portrait-about.jpg` | About ("Über mich") | 4/5            | More personal/candid "in my world" shot (workspace / Alt-Treptow). Same warm grade as the hero. |

Exported as JPG (or swap the extension in the two `<Photo src=…>` references).
