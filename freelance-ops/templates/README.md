# Client Document Templates

Reusable templates for website projects.

Recommended order:

1. `01-discovery-questionnaire.md`
2. `02-proposal-template.md`
3. `03-service-agreement-template.md`
4. `04-deposit-invoice-email.md`
5. `05-balance-invoice-email.md`
6. `06-handover-doc.md`

Optional:

- `07-change-order.md`
- `08-testimonial-request.md`

Notes:

- Replace every `[placeholder]` before sending.
- Keep project scope, revision rounds, payment gates and exclusions explicit.
- For contracts and legal wording, use this as a working draft and have a qualified
  professional review it before relying on it.
